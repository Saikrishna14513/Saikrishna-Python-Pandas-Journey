# Python Pandas Learning Journey

This repository documents my journey of learning Python Pandas for Data Analyst roles.  
It includes key concepts, practical examples, and mini projects.

---

## 📌 Topics Covered

### 1. Loading Data
- `pd.read_csv`, `pd.read_excel`  
- Inspecting data using `head()`, `tail()`, `info()`

### 2. Basic DataFrame Operations
- Viewing columns, selecting rows, slicing  
- Adding or dropping columns  

### 3. Filtering & Sorting
- Filtering rows with conditions  
- Sorting by one or multiple columns  

### 4. Aggregations
- Using `groupby()`, `sum()`, `mean()`, `count()`  

### 5. Merging & Joining
- Combine data from multiple DataFrames using `merge()`  

### 6. Applying Functions & Lambda
- Apply functions to columns or rows  
- Use `lambda` for simple calculations  

---

##  Goal
To master Python Pandas for data manipulation and analysis on real datasets.
